# Newsera

A news aggregator platform.

## Project Structure

This monorepo contains the following components:

- **`mobile-app/`**: React Native Expo app for end-users.
- **`rss-engine/`**: Node.js service responsible for RSS ingestion and processing.
- **`admin-panel/`**: React web dashboard for managing the platform.
- **`docs/`**: Project documentation.
