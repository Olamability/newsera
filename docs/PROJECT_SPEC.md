# News Aggregator Platform

## Goal
A mobile app that aggregates news and drives traffic back to publishers.

## Core Features
- Featured image + title + snippet
- Article detail screen
- Category-based browsing
- External link to full article

## Systems
- Mobile app (React Native Expo)
- Admin panel (publisher management)
- RSS ingestion engine
- Supabase backend

## Auth Rules
- Users must login to bookmark
- Anonymous users can browse

## Content Rules
- No full article display
- Always link back to source
